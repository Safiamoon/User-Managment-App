# ReactJs Routing with CRUD APPLICATION

This app is developed for learning pupose.


## How to run

1. Clone the project from git.
2. Run `yarn install`.
4. Run `json-server --watch db.json --port 3001`
3. Run `yarn start`.

Application will start at http://localhost:3000.
